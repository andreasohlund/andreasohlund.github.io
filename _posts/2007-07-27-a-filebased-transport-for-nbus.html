>While taking Udi:s Nbus for a spin I didn't want to install MSMQ so I've implemented a filebased transport and sub. storage. You can download it from here.

Updated configuration files for Udi:s ServerRunner and ClientRunner is included in the download. In order to get the tests running:

1. Change the "MessageDirectory" and "StorageDirectory" to some directory on your own machine.

2. Add references to NBus.Unicast.Transport.File and NBus.Unicast.Subscriptions.File(or copy the dll:s) from ServerRunner and ClientRunner

Feedback is welcome!
